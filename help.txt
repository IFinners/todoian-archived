  Taskian Command Helper

  Help Information:
    Any Aliases Listed i.e. 'a' and 'add' Will Work For All Additional Examples of The Command Even If Not Stated
    [num] = Any Task Number
    [sub-num] = Any Subtask Number
    {} = Optional Extra Command That Sets Values Directly From the Command

  Viewing Tasklists:
  
    'ls' or 'list' - Shows today's task alongside Overdue tasks (if any) and Tomorrow's tasks (if any)
    'ls t' or 'ls today' - Shows only Today's tasks
    'ls o' or 'ls overdue' - Shows only Overdue tasks
    'ls tm' or 'ls tomorrow' - Shows only Tomorrow's tasks
    'ls f' or 'ls future' - Shows only tasks with a due date later than tomorrow
    'ls a' or 'ls all' - Shows all categories of tasks (Overdue, Today, Tomorrow, Future)

  Task Operations:

    'a' or 'add' - Prompts for a new task to add to the task list with a due date of today
    'a New Task Description {Date} {Repeat}'- Adds a new task with an optional due date and repeat directly from the command
    'rm [num]' or 'remove [num]' - Moves a task into the deleted cache awaiting deletion upon exiting the program
    'rm a' or 'rm all' - Which moves all of the tasks into the deleted cache
    'c [num]' or 'complete [num]'- Moves a task to the completed cache or, if it has a repeat, changes the due date appropriately
    'e [num]' or 'edit [num]' - Prompts for new description for the chosen task
    'e [num] New Description' - Sets the new description directly from the command
    'cd [num]' or 'change-date [num]' - Prompts for a new due date of the chosen task
    'cd [num] YYYY-MM-DD' - Sets the chosen tasks date directly from the command
    'ar [num]' or 'add-repeat [num]' - Prompts for a repeat length for the chosen task
    'ar [num] [number]' - Sets the repeat directly from the command
    'rr [num]' or 'remove-repeat [num]' - Removes the repeat from the chosen task

  Subtask Operations:

    's [num]' or 'subtask [num]' - Prompts for a subtask for the chosen task
    's [num] Substask Description' - Adds the subtask directly from the command
    'cs [num] [sub-num]' or 'complete-subtask [num] [sub-num]' - Marks chosen subtask as complete
    'rs [num] [sub-num]' or 'remove-subtask [num] [sub-num]' - Permanently removes the chosen subtask
    'es [num] [sub-num]' or 'edit-subtask [num] [sub-num]' - Prompts for a new subtask description
    'es [num] [sub-num] New Subtask Description' - Updates the chosen subtask's description directly from the command

  Program Operations:

    'u' or 'undo' - Undoes the last task deletion
    'uc' or 'uncheck' - Undoes the last task completion
    'h' or 'help' - Shows Taskian's Command Helper
